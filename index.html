📖 How to Play

🔧
Place Gates
Drag logic gates from the toolbox onto the breadboard. Gates compute outputs from inputs.
🔌
Draw Wires
Click an output pin (right side) then click an input pin (left side) to connect them.
📊
Match the Truth Table
Your circuit must produce the correct output for every input combination.
▶️
Run Simulation
Press RUN to test your circuit. Green ✓ = correct, Red ✗ = wrong.
💡
Use Hints
Stuck? Click Hint for progressive clues. First hint is free!
⭐
Earn Stars
Fewer gates = more stars. ★★★ for optimal, ★★ for good, ★ for any solution.
